# Dramione Fanfiction Analysis

This notebook analyzes Dramione fanfics from AO3 to find underrated gems and understand fandom trends.

## Files
- `dramione_underrated_gems_analysis.ipynb` - Main analysis notebook
- `ao3_dramione.csv` - Dataset (if you're including it)
- `requirements.txt` - Python packages needed

## To run locally
pip install -r requirements.txt
jupyter notebook dramione_analysis.ipynb
