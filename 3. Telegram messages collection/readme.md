# Telegram Message Collector

A Python script to collect messages from Telegram channels and groups automatically.

## Features

- 📱 Collect messages from multiple Telegram chats
- ⏰ Track messages since last run automatically
- 💾 Save messages to JSON format
- 🚀 Two modes: one-time collection from last time or 24 hours
- 🌐 Support for channels, groups, and private chats
- 🔒 Secure credential management via config file

## Prerequisites

- Python 3.7+
- Telegram API credentials (api_id and api_hash)
- Telegram account (for user client method)


## Output format

[
  {
    "chat": "t.me/channel/message_id",
    "date": "2026-01-01T09:00:00+00:00",
    "text": "Hello world!"
  }
